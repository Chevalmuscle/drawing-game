# Drawing Game

Goal: A game where you and your friends submit words and then have to guess them by drawing them. With the help of p5.js, socket.io, nodejs.

## Installation

0. Install node & npm
1. Clone the repo
2. `npm install`

## Run

`npm start` and open the localhost url printed in the console

## Tools used

- p5.js
- socket.io
- bootstrap
